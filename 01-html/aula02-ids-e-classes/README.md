# Aula 02 — IDs, Classes e Especificação de Tags

Nesta aula aprendi:

- A diferença entre **id** e **class** no HTML
- Como usar CSS interno para estilizar elementos
- Como selecionar elementos usando:
  - #id (seleciona um elemento único)
  - .class (seleciona um ou vários elementos)
- Quais IDs devem ser usados para elementos únicos
- Quais classes são ideais para aplicar o mesmo estilo em vários elementos
