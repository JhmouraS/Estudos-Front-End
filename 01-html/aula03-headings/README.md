# Aula 03 — Hierarquia de Headings (H1 a H6)

Nesta aula aprendi:

- A diferença entre os headings (<h1> até <h6>)
- Que o número do heading indica hierarquia, não apenas tamanho
- <h1> deve ser o título mais importante da página
- <h2> a <h6> são subtítulos que organizam o conteúdo
- Boa hierarquia ajuda na acessibilidade e no SEO
