# Aula 05 — Tags Semânticas Modernas

Nesta aula aprendi:

- A usar **tags semânticas** que ajudam a estruturar melhor o conteúdo da página:
  - `<header>` — cabeçalho da página
  - `<section>` — seção de conteúdo relacionada
  - `<div>` — elemento genérico para organizar blocos
